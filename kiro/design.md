## Design Philosophy
- Game as a learning medium, not entertainment
- Voice-first and visual-first interaction
- Minimal text for low-literacy users
- Real-life financial decision simulation

## Key Screens
1. Home Screen
   - Start Game
   - XP, Level, Resilience Score
   - Voice assistant access

2. Scenario Screen
   - Voice narration of financial situation
   - Icon-based decision choices
   - Simple visuals and large buttons

3. Feedback Screen
   - Rewards or penalties
   - XP / badge updates
   - Voice explanation of outcome

4. Progress Dashboard
   - Visual progress indicators
   - Achievements and scores

5. Offline Mode
   - Offline indicator
   - Auto-sync when online

## Architecture Overview
- Mobile App (Frontend)
- Game & Financial Logic Engine
- AI Voice Guidance Layer
- Backend Services
- Offline-first Data Storage
