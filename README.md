She Leads Finance

## Overview
She Leads Finance is a voice-guided financial decision-making game designed to help women build confidence in budgeting, savings, and digital financial practices through real-life simulations.

## Key Features
- Game-based financial learning
- Voice-guided scenarios
- Budgeting and savings challenges
- Digital finance and fraud awareness
- Resilience Score for financial preparedness
- Offline-first support

## Technology Stack
- Frontend: Flutter / React Native
- Backend: Node.js / FastAPI
- AI: LLM APIs for guidance and narration
- Database: Firebase / PostgreSQL
- Offline Storage: Local device storage
- Hosting: Firebase / AWS / Render

## Why It Matters
- Encourages safe financial behavior
- Builds long-term financial confidence
- Accessible to rural and low-literacy users
- Focuses on behavior change, not theory

## Future Scope
- Multilingual support
- Additional financial scenarios
- Integration with formal financial services
